# Packmann

Ein einfacher Pacman-Klon als Web-App. Zum Starten:

```bash
npm install
npm run dev
```

## Steuerung

- Pfeiltasten oder WASD: Bewegung
- Leertaste: Pause
- R: Neustart
- M: Musik an/aus
- Shift+D: Debug-Modus

## Features

- Lokaler Highscore via localStorage
- Touch-Steuerung für mobile Geräte
- Einfacher Sound mit Howler.js
- Debug-Overlay
